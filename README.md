# Mingle2 - Free Online Dating Site & Chat App

## Project Overview
Mingle2 is a free online dating site and chat app designed for singles to find potential matches, make new friends, and connect with others in a safe and supportive environment. The platform features a user-friendly interface, allowing users to sign up, log in, and communicate effortlessly.

## Installation
To run the Mingle2 application locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Change directory:**
   ```bash
   cd mingle2
   ```

3. **Open the HTML files in your browser:**
   - You can open `index.html` directly in any web browser to view the application.

No additional installation of dependencies is required since the project uses CDN links for Bootstrap and Font Awesome.

## Usage
- Access the main page by opening `index.html`.
- Users can register a new account via the `register.html` file, log in through `login.html`, and explore the site's features.

### Registration
1. Fill in the required fields including username, email, password, gender, and preferences.
2. Click on "Create Account".

### Login
1. Enter your email and password.
2. Click on "Login" to access your account.

## Features
- **Free Membership:** Users can sign up and interact with other members without any charges.
- **Safe & Secure Environment:** The platform implements strict monitoring of profiles to ensure user safety.
- **Sign Up & Find Matches Quickly:** Users can quickly register and start finding potential matches within minutes.
- **Responsive Design:** The application is designed to work well on both desktop and mobile devices.

## Dependencies
The project relies on the following external libraries:
- Bootstrap (for CSS styling): [Bootstrap CDN](https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css)
- Font Awesome (for icons): [Font Awesome CDN](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css)
- Tailwind CSS (for enhanced styles in login and register pages): [Tailwind CSS CDN](https://cdn.tailwindcss.com)

## Project Structure
The project has the following structure:

```
mingle2/
├── index.html        # Main landing page
├── login.html        # User login page
├── register.html     # User registration page
├── styles.css        # Custom styles for the application
└── script.js         # JavaScript for interactivity
```

### Description of Key Files
- **index.html:** The landing page where users can sign up and see the features of Mingle2.
- **login.html:** The page for users to log into their existing accounts.
- **register.html:** The registration form for new users to create an account.
- **styles.css:** Contains custom styles for the application, enhancing the overall appearance.
- **script.js:** Handles user interactions, including form submissions and animations.
